# UI UX Pro Max

An AI skill that provides design intelligence for building professional UI/UX across multiple platforms and frameworks.

## How to Use This Skill

Use this skill when the user requests any of the following:
- "Build a landing page"
- "Create a pricing card"
- "Choose a style/color/font"
- "Review this page for UX issues"

### Step 1: Generate Design System (REQUIRED)

**Always start with `--design-system`** to get comprehensive recommendations with reasoning before writing any UI code:

```bash
python3 ~/.agents/skills/05_marketing_design/ui-ux-pro-max/scripts/search.py "<product_type> <industry> <keywords>" --design-system [-p "Project Name"]
```

**Example:**
```bash
python3 ~/.agents/skills/05_marketing_design/ui-ux-pro-max/scripts/search.py "fintech banking app dark mode" --design-system -p "MyBank"
```

This command will output a complete design system including:
- Pattern, style, colors, typography, effects
- Anti-patterns to avoid (very important to kill AI vibe)
- Pre-delivery checklist

### Step 2: Supplement with Detailed Searches (as needed)

After getting the design system, use domain searches to get additional details:

```bash
python3 ~/.agents/skills/05_marketing_design/ui-ux-pro-max/scripts/search.py "<keyword>" --domain <domain>
```

| Need | Domain | Example |
|------|--------|---------|
| More style options | `style` | `--domain style "glassmorphism dark"` |
| Color palettes | `color` | `--domain color "entertainment vibrant"` |
| Font pairings | `typography` | `--domain typography "playful modern"` |
| Chart recommendations | `chart` | `--domain chart "real-time dashboard"` |
| UX best practices | `ux` | `--domain ux "animation accessibility"` |
| Landing structure | `landing` | `--domain landing "hero social-proof"` |

### Step 3: Implement the Design

Synthesize the design system generated by the python script and strictly follow its guidelines. Pay special attention to the "Avoid (Anti-patterns)" section to ensure the result looks human-crafted and premium.

## The Anti-AI-Vibe Premium Protocol (Anonymous Framework)

To ensure all UI generations look human-coded, premium, and unique, you must map the project to one of these **4 Design Archetypes** based on the industry and target audience. Do NOT copy a single style or cite third-party brand names.

### 1. Cybernetic Tech & High-Density Flat Layouts (DevTools / Systems / Tools)
*   **Visual Style:** Flat, clinical, high contrast, grid-focused, zero clutter.
*   **Typography:** Strict geometric sans-serif paired with sharp, high-density monospace badges.
*   **Colors & Outlines:** Faint transparent borders (6-10% white opacity) or pure gray lines. Grid separation layouts utilizing parent backgrounds as dividers.
*   **Core Aesthetic:** Clean, highly structured technical hubs.

### 2. Neo-Editorial & Asymmetrical Typographic Contrast (Studios / Portfolios / Creative Pages)
*   **Visual Style:** Asymmetric layouts, generous breathing room, high-contrast serif/sans typography clashing, soft glassmorphic panels.
*   **Typography:** Clash between high-precision sans-serif and elegant italicized editorial serifs.
*   **Colors & Outlines:** Premium off-white/cream background with deep charcoal text, or rich dark-slate bases. Outlines are created using fine rings rather than vector strokes.

### 3. Kinetic Volume & Debossed Dark Dimensions (Creative Portals / Immersive Sites)
*   **Visual Style:** Organic dark surfaces, cursor-following canvas effects, glowing borders, active transition curves.
*   **Typography:** Bold geometric sans-serif headings with tight letter-spacings.
*   **Colors & Outlines:** Absolute pitch-black bases, deep dark grey cards, separated by faint dark rings. Action buttons use debossed inner glows for physical tactility.

### 4. Fluid Enterprise & Dashboard Grids (SaaS / Banking / Corporate Dashboards)
*   **Visual Style:** Highly structured data grids, legible charts/metric boxes, clean navigation feeds, responsive blocks.
*   **Typography:** Serious, highly readable system sans-serif fonts.
*   **Colors & Outlines:** Trustworthy deep corporate blues/greens, clean transparent dark blue overlays with backdrop blurs.
