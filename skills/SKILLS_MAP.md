# 🗺️ Agents Skills Map

This file provides a map of the categorized skills available in this directory. Each category is designed to help agents find relevant tools without context overload.

## 📂 Categories

### 🛠️ 01_core
Essential orchestration and system skills.
- `chief-of-staff`: The primary orchestrator for multi-agent workflows.
- `agent-protocol`: Communication rules between agents.
- `command-guide`: Help selecting the right commands and tools.
- `context-engine`: Management of company/project context.
- `decision-logger`: Tracking and reviewing strategic decisions.
- `setup`, `run`, `loop`, `resume`, `eval`: Experimentation and optimization tools.

### 💻 02_development
Full-stack engineering, architecture, and testing.
- `claude-api`: Anthropic SDK integration and optimization.
- `web-artifacts-builder`: Building React/Tailwind/shadcn artifacts.
- `mcp-builder`: Model Context Protocol server development.
- `senior-fullstack`, `senior-backend`, `senior-frontend`, `senior-architect`.
- `tdd-guide`, `playwright-pro`, `api-test-suite-builder`.
- `sql-database-assistant`, `database-designer`, `monorepo-navigator`.
- `focused-fix`, `fix`, `generate`, `init`, `coverage`, `report`.

### 🧠 03_ai_ml
Data science, machine learning, and AI engineering.
- `senior-ml-engineer`, `senior-prompt-engineer`.
- `rag-architect`, `ai-security`, `ai-seo`.
- `senior-data-scientist`, `senior-data-engineer`, `data-quality-auditor`.
- `llm-cost-optimizer`, `llm-wiki`.

### 💼 04_business_ops
Product management, SaaS operations, and C-suite advisory.
- `senior-pm`, `product-manager-toolkit`, `product-discovery`, `product-analytics`.
- `saas-scaffolder`, `saas-metrics-coach`, `revenue-operations`.
- `ceo-advisor`, `cto-advisor`, `cfo-advisor`, `coo-advisor`, `cpo-advisor`, `cro-advisor`.
- `jira-expert`, `confluence-expert`, `scrum-master`.
- `company-os`, `strategic-alignment`, `culture-architect`.

### 📣 05_marketing_design
Growth, marketing, and high-end web design.
- `marketing-skills`, `seo-audit`, `content-production`.
- `epic-design`, `landing-page-generator`, `canvas-design`.
- `page-cro`, `signup-flow-cro`, `onboarding-cro`, `form-cro`.
- `email-sequence`, `social-content`, `x-twitter-growth`.
- `brand-guidelines`, `copywriting`, `copy-editing`.

### 🛡️ 06_devops_infra
Infrastructure, security, and cloud management.
- `senior-devops`, `senior-security`, `senior-secops`.
- `terraform-patterns`, `docker-development`, `ci-cd-pipeline-builder`.
