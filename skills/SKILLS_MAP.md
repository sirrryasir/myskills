# 🗺️ Agents Skills Map

This file provides a map of the categorized skills available in this directory. Each category is designed to help agents find relevant tools without context overload.

## 📂 Categories

### 🛠️ 01_core
Essential orchestration and system skills.
- `chief-of-staff`: The primary orchestrator for multi-agent workflows.
- `accidental-data-loss-prevention`: Safety guardrails for destructive commands.
- `skill-repair`: Fix and re-install failed agent skills.

### 💻 02_development
Full-stack engineering, architecture, and testing.
- `claude-api`: Anthropic SDK integration and optimization.
- `web-artifacts-builder`: Building React/Tailwind/shadcn artifacts.
- `mcp-builder`: Model Context Protocol server development.
- `managing-python-dependencies`: Virtualenv and dependency management.
- `tailwind`: Tailwind CSS v4.2 browser-runtime patterns.
- `senior-backend`, `senior-frontend`, `senior-architect`.
- `playwright-pro`: E2E Testing.

### 🧠 03_ai_ml
Data science, machine learning, and AI engineering.
- `senior-prompt-engineer`: Advanced prompting and LLM optimization.
- `rag-architect`: Retrieval-Augmented Generation design.
- `ml-best-practices`: Core machine learning workflows.

### 💼 04_business_ops
Product management and strategic planning.
- `senior-pm`: Product strategy and management.

### 📣 05_marketing_design
Growth, marketing, and high-end web design.
- `epic-design`: High-end UI/UX design principles.
- `animejs`, `gsap`, `css-animations`: High-end web animations.
- `website-to-hyperframes`: Website to video capture.

### 🛡️ 06_devops_infra
Infrastructure, security, and cloud management.
- `senior-devops`: DevOps and CI/CD operations.
- `senior-security`: Security audits and best practices.
- `docker-development`: Containerization.
- `ci-cd-pipeline-builder`: Pipeline automation.

### 🧰 07_specialized_tools
Document processing and utility tools.
- `hyperframes`, `hyperframes-cli`, `hyperframes-media`, `hyperframes-registry`: Video production.
- `remotion-to-hyperframes`, `three`, `waapi`, `lottie`.
- `browser-automation`: Web scraping and automation.
- `pdf`: PDF document handler.
