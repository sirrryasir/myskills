# Branding Reference Guidelines

This reference documents the specific visual styling, color palettes, and fonts for each core branding option supported by the `video-as-code` skill.

---

## 🎨 Branding Options

### 1. 💼 Yasir Hassan (yaasir.dev)
* **Aesthetic**: Premium Product-Engineer & Full-Stack Developer. Warm graphite panels, elegant light-weight highlights, and high-readability developer-education tools.
* **Background Grid**: OLED Black (`#060606`) with static dark charcoal borders (`#121212`) and thin 1px lines.
* **Pastel Accent Palette**:
  - Main Active highlight: `#7faae4` (Slate Blue) or `#ebd49b` (Soft Amber)
  - Success / Active State: `#9bc3a5` (Sage Green)
  - String / Code literal: `#e05a47` (Rose Red)
  - Comment: `#626262` (Deep Gray)
* **Fonts**:
  - Primary Headers: `Inter`, -apple-system, sans-serif (lightweight, e.g. `200` or `300` weight)
  - Subtitles & Active Captions: `Inter`
  - Code blocks: `JetBrains Mono` or `Geist Mono`

### 2. ⚡ Vercel / Next.js
* **Aesthetic**: Strict monochrome, sharp high-contrast minimalism, publication-level editorial styling.
* **Background Grid**: Pitch black (`#020202`) with razor-sharp slate-black blueprints (`#111111`).
* **Accent Palette**: Pure white (`#ffffff`), zinc-grey (`rgba(255,255,255,0.45)`), and dark carbon charcoal (`#09090b`).
* **Fonts**:
  - Primary Headers: **Fraunces**, Georgia, serif (expressive, low-weight, normal or italicized, e.g. `fontWeight: 400`). This clashes serif elegance with clean code blocks.
  - Subtitles & Active Captions: `Inter`
  - Code blocks: `Geist Mono`

### 3. 🎓 Ardaykaab Academy
* **Aesthetic**: Whiteboard Excalidraw style with hand-drawn structural lines and MacOS clinical window borders. Perfect for programming concepts.
* **Background Grid**: Charcoal blueprint lines with high-density grid structures.
* **Accent Palette**: Minimalist light pastels (sages, peaches, light blues) on graphite backgrounds.
* **Fonts**:
  - Primary Headers: `Outfit` or `Inter`
  - Monospace code: `JetBrains Mono`
